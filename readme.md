### Instalowanie bibliotek używanych przez program
```bash
pip3 install -r requirements.txt
```
### Uruchomienie programu
```bash
python3 main.py -i scieżka_do_pliku_wejścia -o ścieżka_do_pliku_wyjścia --email-subject "Temat maila" --nip-email ścieżka_do_pliku_z_bazą_email #(format_csv)
```
### Listowanie arguentów dostępnych do użycia z programem
```bash
python3 main.py -h
# lub
python3 main.py --help
```
